Política de Exclusão de Dados - AeroProvas

Como solicitar a exclusão dos seus dados

Para solicitar a exclusão dos seus dados do AeroProvas, siga estas etapas:

  1. Envie um email para contato@aeroprovas.com.br
  2. No assunto, escreva: "Solicitação de exclusão de dados - AeroProvas"
  3. No corpo do email, inclua:
    • Seu email
    • Data aproximada de instalação do app

Dados que serão excluídos

Após sua solicitação, excluiremos:

  • Dados de notificações push (OneSignal)
  • Identificadores do dispositivo
  • Quaisquer dados de uso coletados

Prazo para exclusão

Processaremos sua solicitação em até 30 dias úteis. Você receberá uma confirmação por email quando a exclusão for concluída.

Contato

Em caso de dúvidas sobre a exclusão de dados, entre em contato através do email: contato@aeroprovas.com.br
